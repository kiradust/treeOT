# Tree Wasserstein distance with weight training
This is the demo code for the [Tree Wasserstein distance](https://arxiv.org/abs/2206.12116).

## Requirements
Install requirements.
```
sudo pip install -r requirement.txt
```

## Run
Run example.py
```
python example.py
```
